# 🎓 WE School Website

A modern and responsive multi-page website for **WE School of Applied Technology**, built using **HTML** and **CSS**.

## 🚀 Features

* Modern and responsive design
* Multi-page website
* Home Page
* About Page
* Blog Page
* Booking Page
* Registration Page
* Contact Page
* Clean UI
* Responsive Layout
* Organized Project Structure

## 🛠️ Technologies Used

* HTML5
* CSS3
